# Product Context

## Problem
Scientists need professional figures for papers, posters, and presentations. General-purpose design tools (Illustrator, PowerPoint) have steep learning curves and lack biology-specific icons. BioRender solves this with a domain-specific editor and asset library.

## Target Users
Life-science researchers, students, educators, and illustrators who need accurate scientific visuals.

## Key Differentiators to Retain
- Domain-specific icon library (cells, molecules, organisms, lab equipment).
- Template-driven workflow for common figure types.
- Simple, web-based UI with less friction than desktop design software.
- Export formats suitable for journals (TIFF/PNG/PDF with transparent backgrounds).

## User Experience Goals
- Drag-and-drop simplicity.
- Fast canvas interactions (60 fps pan/zoom).
- Clear organization: gallery, editor, templates.
- Minimal onboarding; start from template or blank canvas.
