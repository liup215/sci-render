# SciRender

A browser-based scientific illustration editor inspired by BioRender. Built with React, TypeScript, Vite, and Konva.

## Features

- Canvas editor with zoom, pan, grid, and alignment guides
- Shape tools: rectangle, circle, line, text
- Multi-slide panel
- Selection, multi-selection, transforms, and snapping
- Toolbar alignment actions
- Keyboard shortcuts for tools and editing actions
- Export active slide to PNG

## Tech Stack

- React 19 + TypeScript 5
- Vite 8
- Konva 10 + react-konva 19
- Zustand 5

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
